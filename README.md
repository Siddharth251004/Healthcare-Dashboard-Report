# Healthcare-Dashboard-Report
Developed an interactive Healthcare Dashboard in Power BI using Excel data to analyze hospital operations, patient records, billing performance, and diagnosis trends. The dashboard provides insights into bed occupancy, doctor feedback, diagnosis distribution, billing vs insurance amounts, and patient treatment details through dynamic filters and visualizations. This project demonstrates skills in Power BI, DAX, data modeling, data cleaning, and healthcare data analytics.

Key Features
• Patient-wise filtering and date range selection
• Bed Occupancy Analysis (Private, General, ICU)
• Doctor Feedback Volume Tracking
• Diagnosis Type Distribution
• Billing Amount vs Health Insurance Amount Comparison
• KPI Cards for Admit Date, Discharge Date, Follow-up Date, and Total Billing Amount
• Interactive and user-friendly dashboard design
Tools Used
• Power BI
• Microsoft Excel
• Data Visulization
